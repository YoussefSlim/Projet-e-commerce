## Installation and Setup Instructions

```
git clone git@github.com:YoussefSlim/Projet-e-commerce.git
yarn
```

## Running the app

To get the app up and running (and really see if it worked), run:

```
yarn start
```

## To Lint

```
yarn lint
```

### To Fix Lint

```
yarn lint-fix
```

## To Build

```
yarn build
```
